首页显示”阅读全文”

当遇到这样的需求：在首页不想大篇幅显示文章内容时，只想显示部分，此时可以如下配置：

在themes/next(主题名称)/_config.yml中修改或添加auto_excerpt，将enable设置为true打开，length属性为需要显示的内容长度

此时距离成功还有最后一步，当修改上面的auto_excert之后，是显示了部分内容，但是排版是有问题的，上面的注释已经说明了问题，需要在想要任何显示部分的地方加上<!--more-->，此时可以忽略length，再一次部署之后就是想要的结果。
